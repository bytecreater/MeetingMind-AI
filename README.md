MeetingMind AI 🧠🎙️
Multilingual AI Meeting Assistant with Transcription, Translation, Summarization & RAG

MeetingMind AI is an end-to-end AI-powered meeting assistant that transforms meeting recordings into actionable insights. It can transcribe audio/video recordings, process Hindi and English conversations, generate intelligent summaries, extract action items, answer questions about meetings using Retrieval-Augmented Generation (RAG), and export professional reports.

Built with Streamlit, Whisper, Sarvam AI, LangChain, ChromaDB, Mistral AI, and Hugging Face Embeddings.

🚀 Features
🎤 Audio & Video Processing
Upload meeting recordings or provide YouTube links
Extract audio from video files
Support for multiple audio formats
📝 Speech-to-Text
Automatic transcription using OpenAI Whisper
High-quality speech recognition
Timestamp-based transcript generation
🌐 Multilingual Support
Hindi → English translation using Sarvam AI
Support for multilingual meetings
Unified English transcript generation
🤖 AI-Powered Meeting Intelligence
Generate concise meeting summaries
Extract key discussion points
Identify decisions made during meetings
Generate actionable tasks and follow-ups
📚 RAG-Based Meeting Q&A
Ask questions about any uploaded meeting
Semantic search using vector embeddings
Context-aware responses powered by Mistral AI
Retrieve relevant transcript segments instantly
📄 Export Reports
Download meeting summaries as PDF
Export transcripts
Save action items and decisions
